How to use
MazeGen_RecursiveDivision.exe <mazewidth> <mazeheight> <complexity>

Input:
<mazewidth> = integer (10-50) as maze width
<mazeheight> = integer (10-50) as maze height
<complexity> = float (0-0.3) as maze complexity
		0 means the most difficult
		0.3 meanns easy

Output:
# = wall
  = path
F = finish line
S = start position

Example:
Mazegen_RecursiveDivision.exe 20 20 0.1
